# Workshop Exercises

This package is where you build your solutions to the exercises in
[workshop-exercises.md](../../../../workshop-exercises.md).

Each exercise includes a scaffolded test class and names the file it belongs
in (for example, `src/test/java/exercises/SwitchExpressionsTest.java`).
To get started:

1. Copy the scaffold code from the exercise description.
2. In IntelliJ IDEA, select this `exercises` package in the Project view and
   paste. The IDE creates the file with the correct name automatically.
3. Replace the `TODO` sections until the tests pass.

Reference solutions for every exercise are in the
[`solutions`](solutions) subpackage — try the exercise before peeking.
